= torrentsync

Sync torrents to BitTorrent clients such as Deluge.

== Install

* sudo apt-get install ruby-dev g++
* sudo gem install thin sinatra bencode nokogiri openuri choice haml json

== Run Web Server

* bin/torrentsyncweb

== Run Command

* bin/torrentsync --help

== Settings

* ~/.torrentsync/settings.yaml
